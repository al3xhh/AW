WebMusic

WebMusic será una página en la que se podrá compartir tu música con todos los demás, al estilo de la aplicación SoundCloud.
La aplicación tendrá varios tipos de usuarios. El usuario no registrado, el usuario registrado y el usuario premium.
También tendrá un aspecto de red social. A cada canción se le podrá dar una valoración, aún por decidir. Además se podrán poner comentarios en cada canción.
